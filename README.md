Описание
=====
В [википедии](https://ru.wikipedia.org/wiki/Snake_%D0%B8%D0%B3%D1%80%D0%B0) можно найти историю и описание игры. Змейка написанная мной является модификацией той игры. Тут вместо одного уровня несколько уровней. Игра делится на три уровня: Easy, Medium и Hard. При входе в игру откроется меню для выбора одного из уровней.
